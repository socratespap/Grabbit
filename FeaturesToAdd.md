# Popup Customization Features

## 1. Button Visibility Options
Add the ability to control which buttons are visible in popup.html:
- Checkbox for "Copy Selected Tab URLs" button
- Checkbox for "Copy All Tab URLs" button
- Checkbox for "Open Copied Links" button
- When a checkbox is deselected, the corresponding button should not be shown in popup.html

## 2. Default Action on Extension Icon Click
Add the ability to set a default action when the extension icon is clicked:
- Currently, clicking the icon opens popup.html
- Add option to immediately perform one of the following actions without opening the popup:
  - Copy Selected Tab URLs
  - Copy All Tab URLs
  - Open Copied Links

## 3. URL Format Options
Add customization options for URL formatting when copying links:
- Select a default separator for each copied tab link
- Configure number of separators (default is 1)
- Option to include tab titles when copying all tabs (checkbox)
