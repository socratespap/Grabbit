Experimental Features

Configure advanced behavior and experimental functionality.

🔗 Linkify

Automatically converts text that looks like a URL (e.g. https://google.com, www.example.com) into clickable links. Includes support for links inside code blocks.

Recognizes links without prefixes (e.g., "google.com"). ⚠️ Note: This may cause false positives on filenames or version numbers (e.g. app.js).

✨ Duplicate Link Highlighter

Visually highlights links that appear more than once on the page with a colored underline. Different colors are used for each set of duplicate links.

🚫 Exclusion Filters

Add keywords or Regular Expressions to exclude matching links from drag-selection.
Links containing these patterns in the URL will be automatically skipped.

    Add keywords or Regular Expressions to exclude matching links from drag-selection based on their visible text (anchor text, aria-label, title, or image alt).
    Links containing these patterns in the visible text will be automatically skipped.

      ⛔ Disabled Domains

      Prevent Grabbit from running on these domains entirely (e.g., `google.com`, `figma.com`).

        These features are experimental and may change in future updates.
